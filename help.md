# Shortcuts

## Move cursor
- up/down : "up/down"

## Modify shown elements
- collapse/expand (or fold/unfold): "left/right"

## View file
- open text snippet: "enter"

## Move item
- up/down (inside Heading): "super+up/down"
- outdent/indent: "super+left/right"

# Modify item
- create heading: "n, h"
- create text snippet: "n, t"
- rename item: "super+r"
- delete item: "super+d"
